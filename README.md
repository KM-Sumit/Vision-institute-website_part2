# Vision-institute-website# Vision Institute Website

This is a professional website for a coaching institute, built with modern web technologies.

## Features

- **Responsive Design**: Looks great on all devices (desktop, tablet, mobile).
- **Modern UI**: Clean, user-friendly interface with smooth scrolling.
- **Complete Sections**: Includes Home, About, Courses, Contact, and more.

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd vision-institute-website
   ```
3. Open `index.html` in your web browser.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Styling and animations.
- **JavaScript (ES6+)**: Interactivity and features.
- **Bootstrap 5**: Framework for responsive design.
- **Font Awesome**: Icons and symbols.
